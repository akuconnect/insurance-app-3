# Quick Motor Quote - React Version

This is a React.js version of the Quick Motor Quote application, converted from the original HTML/JS/CSS app.

## Features

- Dashboard with various insurance quote options
- Private Car premium calculator (basic implementation)
- Responsive design
- Font Awesome icons
- Google Fonts integration

## Getting Started

1. Navigate to the quote folder:
   ```
   cd d:\code\insurance\testapp\quote
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
